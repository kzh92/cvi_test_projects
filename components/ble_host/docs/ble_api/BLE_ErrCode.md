## Title:  错误码说明



| BLE Stack错误码说明       |              |
| ------------------------- | ------------ |
| BLE_STACK_OK = 0,         | 无错误       |
| BLE_STACK_ERR_NULL=1,     | 参数为空指针 |
| BLE_STACK_ERR_PARAM=2,    | 参数错误     |
| BLE_STACK_ERR_INTERNAL=3, | 内部错误     |
| BLE_STACK_ERR_INIT=4,     | 初始化错误   |
| BLE_STACK_ERR_CONN=5,     | 连接错误     |
| BLE_STACK_ERR_ALREADY=6,  | 已经处理过   |
